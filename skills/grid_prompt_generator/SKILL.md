# 图片提示词生成指南

本 SKILL 对应 `grid_prompt_generator` Agent，支持生成三类中文图片提示词：

1. **角色图片提示词** — 角色外貌与气质
2. **场景图片提示词** — 场景氛围与光线
3. **宫格图提示词** — 多镜头网格拼图

详细模板见 `reference/` 目录。

---

## 角色图片提示词

参考：`reference/character-prompt.md`

### 模板结构
```
[单人全身角色立绘]，[完整人物从头顶到脚底全部进入画面]，[外貌描写]，[服装与鞋子完整可见]，[性格/气质]，[角色定位]，[高质量]，[统一美术风格]，[禁止半身照/近景/裁切/文字水印]
```

### 生成规则
- 以 `appearance`（外貌描述）为核心
- `personality` 决定气质基调（内敛/张扬/神秘等）
- `role` 决定服装和道具风格
- 必须使用中文输出，不要输出英文画质词或英文镜头词
- 必须明确写：单人全身角色立绘、完整人物从头顶到脚底全部进入画面、头发/脸/上半身/双手/腿/脚踝/鞋子都清楚可见
- 横屏项目也必须强制全身构图：人物直立站姿、居中构图、镜头距离足够远、身体上下留有少量空白
- 必须明确负向约束：不要半身照、胸像、头像、近景、特写、坐姿、蹲姿、趴卧、被遮挡、裁切头部/手臂/腿部/脚部/衣摆、多人、文字、签名、水印
- 不要使用“角色肖像”“半身肖像”“特写肖像”等容易诱导半身图的词

## 场景图片提示词

参考：`reference/scene-prompt.md`

### 模板结构
```
[地点]，[时间段]，[光线氛围]，[场景描述]，[电影级场景画面]，[高质量]，[统一美术风格]，[不要文字、签名或水印]
```

### 生成规则
- 以 `location`（地点）为基础
- `time` 决定光线色调（白天/夜晚/黄昏）
- 场景氛围词使用中文，如“氛围浓郁、压抑、温暖、冷冽”等
- 必须包含“电影级场景画面”和“统一美术风格”
- 避免出现文字、签名、水印

## 宫格图提示词

参考：`reference/shot-prompt.md`

### 三种模式

#### 首帧模式 (first_frame)
每个格子 = 一个镜头的起始画面，但必须严格生成用户指定的 `rows x cols` 总格数。

```
[rows x cols 宫格布局]，必须正好 [rows*cols] 个可见画格，统一美术风格，[风格描述]，
格1：[镜头1起始画面]，
格2：[镜头2起始画面]，
格3：[镜头3起始画面]，
...
格N：[起始画面]，
高质量，电影级光影，不要合并画格，不要缺失画格，不要文字或水印
```

#### 首尾帧模式 (first_last)
保持首尾帧节奏感，但仍然必须严格生成用户指定的 `rows x cols` 总格数，不允许偷偷改成 `Nx2`。

```
[rows x cols 宫格布局]，必须正好 [rows*cols] 个可见画格，统一美术风格，[风格描述]，
格1：[起始画面]，
格2：[结束画面]，
格3：[起始画面]，
格4：[结束画面]，
...
高质量，电影感，体现连续动作，不要合并画格，不要缺失画格，不要文字
```

#### 多参考模式 (multi_ref)
所有格子都是同一镜头的不同角度/构图参考，但仍然必须严格生成用户指定的 `rows x cols` 总格数。

```
[rows x cols 宫格布局]，必须正好 [rows*cols] 个可见画格，同一场景不同角度，[风格描述]，
[主体场景描述]，
格1：远景建立环境，
格2：中景突出角色，
格3：近景细节，
格4：戏剧化角度，
...
统一光线和色彩，不要合并画格，不要缺失画格，不要文字
```

### 通用规则
1. 提示词使用**中文**
2. 必须明确写出用户指定的 `rows x cols grid layout`
3. 必须包含“统一美术风格”保持风格统一
4. 必须明确要求“正好 N 个可见画格”
5. 必须明确要求“不要合并画格，不要缺失画格”
6. 避免在格子间出现分割线的描述
7. 尺寸建议：每格 960x540，总图 = 960×cols × 540×rows
8. 当存在参考图映射时，统一使用 `图片1/图片2/...` 指代参考图，不要把它和 `格1/格2/...` 混用
