# Changelog

## 2026-04-30

- 品牌更新为 VeryAI 短剧平台 / VeryAI Shorts，并替换页面 Logo 与浏览器图标。
- AI 服务配置改为开放式配置，支持自定义 provider、Base URL、模型、端点和高级 JSON。
- 支持 OpenAI 兼容接口、通用任务式 API、ComfyUI 工作流 API。
- ComfyUI 支持多个 Base URL，可轮询分发任务到多台后端。
- 新增全局默认模型和项目默认模型，文本、图片、音频、视频可分别选择。
- 图片模型拆分为角色形象、场景图片、镜头图片，适配文生图和图生图工作流。
- 创建项目支持导入 PDF、Word、TXT、MD，并按计划集数分配内容。
- 新增横屏/竖屏成片画幅设置，生成图片和视频时传递对应尺寸参数。
- 分镜、场景图、视频提示词改为中文优先。
- 角色库支持跨集复用、编辑和重新生成。
- 自动生成支持全剧、前 N 集、指定集数，并可生成到分镜、镜头图片、视频生成或视频与语音合成阶段。
- 自动生成任务支持进度条、任务描述、暂停、继续、终止、重新开始和任务日志。
- 自动生成前检测已有资产，默认继续补缺，只有明确选择后才会重置并重新生成。
- 视频生成支持批量下发镜头任务，提高 ComfyUI/任务式接口并发效率。
- 视频与语音合成会保留原视频声音；没有 TTS 时跳过配音合成，最终拼接优先使用原始有声视频。
- 剧集首页改为网格排布，并展示画幅、创建时间和最近修改时间。
- 多处图片预览适配横屏、竖屏和方图，避免缩略图裁切异常。
- 新增后台运行说明和本地敏感数据忽略规则，避免提交 API Key、数据库和生成资产。
